# Desktop

Aplicacao desktop Windows do CRM Autopecas e Servicos.

## Tecnologia Planejada

- Flutter Desktop.

## Status

Pasta reservada para o projeto Flutter. Antes de gerar o aplicativo, confirmar se o Flutter SDK esta instalado e acessivel no terminal.

## Responsabilidades

- Interface do usuario.
- Navegacao.
- Validacoes simples de formulario.
- Consumo da API backend.
- Experiencia de uso para operacao diaria.

Regras de negocio criticas devem ficar no backend.
